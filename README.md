# 🔐 Phishing URL Detector (Flask)

## 📌 Description
A Python Flask web application that analyzes URLs and detects phishing attempts using a scoring-based algorithm.

## 🚀 Features
- URL analysis using multiple security checks  
- Detects suspicious patterns (IP, shortener, symbols, etc.)  
- Calculates phishing score (0–100)  
- Gives result: Safe / Low Risk / Suspicious / Phishing  

## 🛠️ Tech Used
- Python  
- Flask  

## ▶️ How to Run
1. Install dependencies:
       pip install flask 
2. Run the app:
      python app.py
3. Open in browser:
      http://127.0.0.1:5000

This project is for educational purposes and basic phishing detection only.
